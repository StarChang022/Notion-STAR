Sync and Relation

By the way, using Sync and Relation can better unleash the powerful features of Notion.

  • Author
  • STAR Chang

Sync allows you to fully synchronize all content within that block to any page. Similarly, if a user does not have permission to access the source page, they will not be able to see that block.

Here are the steps to use Sync:

  1. Call Synced block
  2. The block will change to a Synced block, which will serve as the source of the synchronization.
  3. Enter any content within the block and click on "Copy and sync." This block will be copied to the clipboard.
  4. Go to any page where you want to paste the block. Now, both sides will be fully synchronized.
