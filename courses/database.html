As a database

One amazing thing about Notion is that it allows you to create Database, which can be used as simple forms or as a database for indexing, similar to Excel.

  • Author
  • STAR Chang

Notion allows you to directly create databases within the editing page, which can be used to organize data and referenced as well, offering both functionality and convenience. Databases in Notion come in several styles, including Table, Board (similar to Trello), Gallery, List, Calendar, and Timeline. You can apply these styles based on your needs and change them anytime as required.

Basic Functions

TAB can be used to switch between or add/delete other styles.

The toolbar has multiple purposes. Filter allows you to filter the data in the database based on your needs, Sort automatically arranges the data, Magnifier enables keyword search, and the More icon provides access to advanced settings.

There are three ways to add data: the "New button" at the top, the "New row" option below the table, and mouseover on the left side of the table.

At the bottom of each column, you can perform calculations or statistics.

To add a new field in the database, you can directly click the "Plus button" in the Header row or click "Add a property" in the card view.

Clicking on an existing field allows you to edit or perform other actions.

To open a card, hover over it and click "OPEN". Regardless of the style of the database, each data entry is represented as a card.

The way to connecting a Database is as follows

When you create a new Database, regardless of the style you choose, you will see the "Select data source" option. Selecting "New database" will allow you to create a brand new dataset. If you want to connecting an existing Database, you can choose the desired list from the menu at the top.

What is connecting a Database?

"Connecting a Database" means using the same data source in multiple places. In some situations, it is necessary to maintain the same data on different pages. For example, the Sales Department and the Customer Service Department may both need to manage a Products List on their respective pages. Any modifications made to the inventory quantity will be synchronized across the Databases that everyone sees.

What's even better is that Databases can also manage permissions. Suppose a page contains an article, a few images, and an embedded Database. People without permissions to access the Database will only see the article and images.
