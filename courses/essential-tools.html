Call commands

There are several ways to call commands. You can create a new block by typing "/" in a blank area, or you can modify an existing block by hovering over it and clicking the "+" icon. It is important to familiarize yourself with the available commands in order to navigate seamlessly within Notion.

I will explain several commonly used important features.

Basic blocks

  • Text: is used for entering general text.
  • Page: is used to create a new page.
  • To-do list: is used to create a checklist with checkboxes.
  • Heading 1, Heading 2, Heading 3: are used for creating large, medium, and small headings, respectively.
  • Table: is used to create a table (non-database style).
  • Bullet list: is used for creating a bulleted list, similar to what you would find in a blog editor.
  • Numbered list: is used for creating a numbered list, similar to what you would find in a blog editor.
  • Toggle list: is used to create a collapsible and expandable list.
  • Quote: is used for creating a blockquote.
  • Divider: is used to insert a horizontal line.
  • Link to page: is used to create a shortcut that links to an existing page.
  • Callout: is used to create a card-like block with customizable icons, titles, and content.

Media

  • Image: is used to insert an image, and you can adjust its size and alignment.
  • Web bookmark: is used to save a bookmark for a specific webpage.
  • Video: is used to insert a video, such as embedding a YouTube video.
  • Audio: is used to insert an audio file.
  • Code: is used for displaying code in a specific code format.
  • File: is used to upload various types of files, including documents, compressed files, images, and more.

Selecting Text

After entering text, you can select the desired portion of the text and call up the function menu.

Here is an explanation of several commonly used important features.
  1. Switch the function of the selected text block, similar to calling functions using commands.
  2. Link the selected text to a Notion page or an external link.
  3. Expand a discussion thread for the selected text.
  4. Apply formatting options such as bold, italic, underline, strikethrough, code format, and equation format to the selected text.
  5. Change the text color or background color of the selected text.
  6. Mention collaborators or insert shortcuts to Notion pages after the selected text.
  7. Call additional actions.
