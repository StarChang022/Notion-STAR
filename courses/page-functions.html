Page functions

Each Page Functions can be customized and tailored to your specific needs and requirements based on the current situation. You have the flexibility to personalize and adjust various details accordingly.

  • Author
  • STAR Chang

Each page can be independently configured or perform basic actions by clicking the function menu in the top right corner.

Here is an explanation of several commonly used important functions:
  • Style: Adjusts the font of the page, with three options available: Default, Serif, and Mono.
  • Small text: Reduces the font size of the page.
  • Full width: Switches the page between full width and narrow width.
  • Move to: Allows you to specify the location where the page should be moved.
  • Add to Favorites: Adds the page to the Favorites list in the sidebar for quick access.
  • Duplicate: Creates a copy of the page at the same hierarchical level. For example, if you duplicate "1st layer > 2nd layer > 3rd layer Page A" to create Page B, Page B will also be located in the 3rd layer.
  • Undo: Performs the undo function, reverting to the previous action.
  • Page history: Shows the editing history of the page, with automatic system backups. The duration of storage depends on your membership plan.
  • Show deleted pages: Displays pages that have been previously deleted within the page.
  • Delete: Deletes the page.
  • Import: Allows you to import content from external files. Notion supports 12 different formats, including HTML, Markdown, CSV, Word, Google Docs, as well as platforms like Trello and Evernote.
  • Export: Export the page in PDF, HTML, or CSV format. Additional options include selecting whether to include all data or only data specific to the page, displaying all content or excluding files and images, showing subpages, and creating folders for new subpages.
  • Connections: An impressive feature that enables integration with third-party services' APIs. For example, you can use Zapier to connect Google Calendar with Notion forms. Currently, there are 15 third-party services available for integration, with more expected to be added in the future by Notion.
